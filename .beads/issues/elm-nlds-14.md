## Description

Set up `elm-doc-preview` for previewing package documentation locally with hot reloading.

Benefits:
- See documentation changes in real-time
- Catch doc generation errors before publishing
- Preview how docs will look on package.elm-lang.org

## Setup

1. Install: `npm install --save-dev elm-doc-preview`
2. Run: `elm-doc-preview`
3. Open browser to localhost

Alternative: Use online tool at https://elm-doc-preview.netlify.app/ (requires pushing docs.json)
